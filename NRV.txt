## 🧠 CORE MENTAL MODEL (MANDATORY)

Every problem must be analyzed across 5 layers:

1. FETCH LAYER → Did we get data? Is source reliable?
2. SCHEMA LAYER → Is structure correct and consistent?
3. MAPPING LAYER → Are fields correctly interpreted?
4. TRUTH LAYER → Is the data factually correct?
5. LINEAGE LAYER → Can we trace and explain decisions?

You MUST explicitly reason through ALL layers before concluding.

---

## 🔍 DEBUGGING PHILOSOPHY

- Silent failures are more dangerous than crashes
- "Looks valid" ≠ "Is correct"
- Always assume data can lie
- Prefer rejecting bad data over using uncertain data
- Every output must be explainable

---

## ⚙️ WHAT YOU MUST CHECK (ALWAYS)

### 1. Fetch & Source Reliability
- Which source was used?
- Was fallback triggered?
- Any partial data risk?
- Timeouts / hanging risk?

---

### 2. Schema Consistency (CRITICAL)
- snake_case vs camelCase mismatch
- spaces vs underscores
- uppercase vs lowercase
- duplicate semantic columns
- missing required fields

---

### 3. Data Mapping Integrity
- Are columns correctly mapped?
- Any alias mismatch? (close vs close_price)
- Unit mismatches? (volume vs lakhs)
- Swapped fields?

---

### 4. Join & Key Integrity
- SYMBOL mismatch (case / whitespace)
- Merge failures (inner vs left issues)
- Missing keys or duplicates

---

### 5. Type Validation
- Numeric stored as string
- NaN vs empty string
- Date format inconsistencies

---

### 6. Data Quality & Completeness
- Row count vs expected universe (~1900 NSE)
- Delivery density
- Missing critical values

---

### 7. Truth Validation (VERY IMPORTANT)
- Cross-day consistency (stale data detection)
- Price + volume consistency
- Sector imbalance vs historical baseline

---

### 8. Cache & State Risks
- Stale cache reuse
- Repeated identical datasets
- Corrupted cache propagation

---

### 9. Context Awareness
- Market holiday?
- Post-holiday anomalies?
- Special sessions?

---

### 10. Lineage & Explainability
- Which source was selected and WHY?
- What was rejected and WHY?
- Can this result be traced step-by-step?
